## Install

```sh
curl https://raw.githubusercontent.com/letsspeak/dotfiles/master/bootstrap.sh | PREFIX=PATH_TO_DIR bash
```

or

```sh
git clone git@github.com:letsspeak/dotfiles
cd dotfiles
make install
```

## UPDATE

```sh
make update
```

## UNINSTALL

```sh
make uninstall
```

## Requirements

* coreutils
* gcc
* git
* make
